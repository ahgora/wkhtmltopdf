#### Para Converter HTML em PDF 

Um arquivo unico e executavel para usar no repositorio ahgora/alpine-php5-build ou conforme sua necessidades



#### Para usar siga os passos abaixo :


wget [ https://github.com/ahgora/wkhtmltopdf/raw/master/wkhtmltopdf ]( https://github.com/ahgora/wkhtmltopdf/raw/master/wkhtmltopdf)

chmod +x wkhtmltopdf

mv wkhtmltopdf /usr/bin/
